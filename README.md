# RISC-V CPU Emulator & Cache Simulator

Высокопроизводительный эмулятор процессора на базе архитектуры **RISC-V (RV32I)**, написанный на Java. Проект включает в себя полноценный цикл обработки инструкций (Fetch-Decode-Execute), многоуровневую систему кэширования и встроенный ассемблер.

## 🚀 Основные возможности

* **Эмуляция RV32I:** Поддержка основных типов инструкций: R-type, I-type, B-type, U-type и др.
* **Гибкая система кэширования:**
    * 4-way set associative cache (4-канальный частично-ассоциативный кэш).
    * Реализация двух политик вытеснения: **LRU** (Least Recently Used) и **Bit-pLRU** (Pseudo-LRU).
    * Раздельный сбор статистики попаданий (Hit Rate) для инструкций и данных.
* **Встроенный Ассемблер:** Трансляция человекочитаемого кода RISC-V в бинарный машинный код (Hex/Binary).
* **Симуляция RAM:** Виртуальное адресное пространство объемом 256 КБ (18-битная адресация).



## 🛠 Технический стек

* **Language:** Java 17+
* **Architecture:** RISC-V (Base Integer Instruction Set)
* **Key Concepts:** Binary manipulation, Memory Management, Cache Replacement Policies, Assembler Parsing.

## 🏗 Архитектура проекта

### 1. CPU (`Cpu.java`)
Ядро эмулятора. Содержит блок регистров (32 РОН), программный счетчик (PC) и логику выполнения команд. 
* **Точка входа:** По умолчанию `0x20000` (`1 << 17`).
* **Механизм:** Реализует чтение инструкций через кэш и их декодирование с помощью `AssemblerParser`.

### 2. Cache System (`Cache.java`)
Симулирует поведение аппаратного кэша для анализа эффективности алгоритмов.
* **Параметры:** Размер строки 32 байта, ассоциативность 4-way.
* **Алгоритмы:**
    * **LRU:** Реализован через систему счетчиков приоритета внутри каждого набора (set).
    * **Bit-pLRU:** Эффективная аппроксимация, использующая биты доступа для минимизации вычислительных затрат на вытеснение.

### 3. RAM (`Ram.java`)
Эмулирует основную память. Поддерживает загрузку ассемблерного файла и выдачу данных целыми кэш-линиями (32 байта) для имитации работы реальных шин данных.

### 4. Assembler Parser (`AssemblerParser.java`)
Модуль для перевода мнемоник RISC-V в машинный код. Обрабатывает:
* Регистровые имена (`ra`, `sp`, `t0`...).
* Различные форматы чисел (Hex, Dec).
* Формирование битовых масок для инструкций типа R, I, B, U.
